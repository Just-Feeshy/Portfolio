Project Overview

Spoopy Environment (Spoopy Envi. for short) is a surface-level, lightweight framework designed to simplify game development, serving as a higher abstraction layer.

Created as a high school capstone project, it later evolved into a versatile graphics framework recognized for early Vulkan/Metal integration.
