Dinomite

Inspired by the Chrome Dinosaur Game.

Website

Controls

  • W / Up Arrow - Jump
  • S / Down Arrow - Increase Gravity
  • Enter - Accept Option In A Menu Or Pause Game
  • Backspace / Escape - Exit Menu
  • "+" - Increase Volume
  • "-" - Decrease Volume

Project Overview

A game I made in 10 hours during my senior year of high school for a show-and-tell presentation in collaboration with other students in my class at the time.

I chose to write this game using the HaxeFlixel framework to get cross-compatibility right out of the box, plus I hate using game engines.
